## Currency Converter
Currency-Converter is a user-friendly web application built using HTML, CSS, and JavaScript. This app leverages real-time data from a currency API to provide accurate and up-to-date exchange rates.

## Tech Stack
1. HTML5
2. CSS3
3. JavaScript
